
# terraform-learning

This repository contains my Terraform learning projects and experiments.

It is used to practise Infrastructure as Code (IaC) using Terraform, mainly focusing on AWS resources and core Terraform concepts.

---

## 🧱 What’s inside

The repository may include Terraform configuration files such as:

* Provider configuration
* EC2 instances or other AWS resources
* Variables and outputs (as needed)
* Terraform state-related setup (not committed)

Files will change frequently as I continue learning.

---

## 🚀 How to use

Typical workflow:

```bash
terraform init
terraform plan
terraform apply
```

To destroy resources:

```bash
terraform destroy
```

---

## ⚠️ Notes

* This is a learning repository
* Files and structure may change at any time
* Sensitive and generated files (like `.tfstate`) are excluded via `.gitignore`
* Not intended for production use

---

## 📚 Purpose

The goal of this repo is to:

* Learn Terraform fundamentals
* Understand AWS infrastructure provisioning
* Practice Infrastructure as Code workflows
* Build confidence with real cloud deployments
